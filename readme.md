# SDL Game Engine

SDL Game Engine is a 2D game engine based on SDL2.


## Setting up SDL Game Engine

Requirements:

 - CMake
 - SDL2
 - SDL2 image
 - SDL2 mixer
 - C++17 or higher

You can use the CMake GUI or you can run the following commands in the commandline:
```
mkdir build
cd build
cmake ..
cmake build .
```
